 Digital Equb System

A modern web-based Ethiopian Equb management system that helps users create and manage Equb groups, register members, make monthly payments, track distributions, and manage the system through an admin dashboard.



## Features

- User Registration & Login
- Secure Authentication
- Create and Manage Equb Groups
- Monthly Payment Tracking
- Distribution Management
- Admin Dashboard
- Responsive Design
- Multi-language Support (English, Amharic, Afaan Oromo)

---

## Technologies Used

### Frontend
- React
- HTML5
- CSS3
- JavaScript
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Bcrypt

 Installation

 Clone the repository

```bash
git clone https://github.com/yab-jej/digital-Equb.git
```

 Go to the project folder

```bash
cd digital-Equb
```

 Install frontend dependencies

```bash
cd equb/equb-client
npm install

 Install backend dependencies

bash
cd ../equb-server
npm install


 Run backend

bash
npm run dev


 Run frontend

bash
npm run dev
